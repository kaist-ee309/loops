### Task Handling Process

- Create an issue for each task
- Assign the responsible person for the issue
- Close the issue once the task is resolved

### Workflow on Kanban Board

- Move the task to WIP (Work in Progress) while working
- Transfer the task to Done after completion
- Maintain clear status visibility for all team members
